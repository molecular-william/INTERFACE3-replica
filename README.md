# INTERFACE3-replica
This repository is an re-implementation of INTERFACE3, a software for analyzing higher order helix-helix interactions in proteins, as a streamlit app.



Larisa Adamian, Ronald Jackups, T.Andrew Binkowski, Jie Liang,
Higher-order Interhelical Spatial Interactions in Membrane Proteins,
Journal of Molecular Biology,
Volume 327, Issue 1,
2003,
Pages 251-272,
ISSN 0022-2836,
https://doi.org/10.1016/S0022-2836(03)00041-X.
(https://www.sciencedirect.com/science/article/pii/S002228360300041X)
